## Vue d’ensemble

Zwii is a database-less (flat-file) CMS that allows you to easily create and manage a web site without any programming knowledge.

### Features

- No Database
- Multi-users
- Multilingual
- 100% responsive
- WYSIWYG editor
- Files manager
- Flexible


**Version incluse :** 12.3.08~ynh1
