**Zwii is a database-less (flat-file) CMS that allows you to easily create and manage a web site without any programming knowledge.**

> NOTE: Zwii is now available in multiple languages.

### Features

- Simple and complete
- No Database
- In your image
- Multi-users
- Multilingual
- 100% responsive
- WYSIWYG editor
- Files manager
- Flexible
- Open source
