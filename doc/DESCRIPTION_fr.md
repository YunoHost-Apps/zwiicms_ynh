Zwii est un CMS sans base de données (fichier plat) qui vous permet de créer et de gérer facilement un site Web sans aucune connaissance en programmation.

### Caractéristiques

- Pas de base de données
- Multi-utilisateurs
- Multilingue
- 100% réactif
- Éditeur WYSIWYG
- Gestionnaire de fichiers
- Flexible
